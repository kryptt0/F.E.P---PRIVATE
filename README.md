# Installation

This will install Git

```bash
sudo apt-get install git
```

This will clone the repository

```bash
git clone https://github.com/kryptt0/F.E.P---PRIVATE
```

Change into  directory F.E.P---PRIVATE

```bash
cd F.E.P---PRIVATE
```

# Update

